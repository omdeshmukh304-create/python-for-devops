# Start Here

Welcome! This is the code for **Python for DevOps + Agentic AI**. Here's how to
get the most out of it.

## How the course runs

It's hybrid — recorded modules you go through at your own pace, plus a live
weekend where we build the bigger pieces together.

- **Recorded (self-paced):** foundations, APIs & JSON, file handling & logs, OOP,
  CLI tools, and the full AI-agent build. Watch these anytime and do the Practice.
- **Live sessions:** we live-code AWS automation and the FastAPI service, demo the
  AI agent, and assemble the capstone — then take your questions. Live recordings
  are added to the LMS afterward, so nothing is missed.

Check the course platform for your cohort's live dates and joining link.

## Before the live weekend

1. Finish [`SETUP.md`](SETUP.md) so your machine is ready.
2. Watch and try the recorded modules **01–05** and **08**.
3. Bring questions — the live sessions are the place to get unstuck.

## The path

Go in order; each module has a short README with the code and a Practice section.

1. [01-python-foundations](01-python-foundations/) — variables, control flow, functions, a system health script
2. [02-apis-and-json](02-apis-and-json/) — requests, JSON, collections, file I/O
3. [03-file-handling-and-logs](03-file-handling-and-logs/) — read logs, count levels, write a summary
4. [04-object-oriented-python](04-object-oriented-python/) — the same analyzer as a class
5. [05-cli-tools-argparse](05-cli-tools-argparse/) — turn it into a real CLI tool
6. [06-aws-automation-boto3](06-aws-automation-boto3/) — read AWS with Boto3 (+ CDK intro) — *live*
7. [07-apis-with-fastapi](07-apis-with-fastapi/) — a DevOps API with FastAPI — *live*
8. [08-ai-agents-for-devops](08-ai-agents-for-devops/) — a local LangGraph + Ollama log agent

Then [`capstone/`](capstone/) to tie it together, and [`guides/`](guides/) for
design thinking, the DevOps mindset, and explaining your work in interviews.

## How to actually learn this

- Don't binge. Watch a module, then do its Practice before moving on.
- Type the code yourself — don't just read it.
- Break things on purpose and fix them. That's the job.
